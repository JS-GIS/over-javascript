## 一 字符串概述

JS中的字符串由双引号或者单引号括起来表示：
```js
var str1 = "字符串1";
var str2 = '字符串2';
```

在ES5及其以后，字符串可以进行换行书写，每行末尾必须以`\`结束。
```js
var str = "aaa\
aa";
console.log(str);          // aaaaa
```

当如果在字符串的值内部直接书写`\n`会表示换行：
```js
var str = "aaa\naa";
console.log(str);          // aaa aa
```

字符串最常见的操作就是拼接：
```js
var str1 = "hello";
var str2 = "world";
console.log(str1 + " " + str2);
```

## 二 字符串的常用属性

```js
var str1 = "hello";
var str2 = "world";

// 长度
console.log(str1.length);       // 5

```
