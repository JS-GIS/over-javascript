## 二 企业级进程管理工具pm2

针对上一节中Node的问题，笔者推荐使用进程管理工具pm2，pm2可以有效的监控服务器状况，并让node服务器自动重启，支持守护进程。  

安装pm2：
```
npm install pm2 -g
```

使用pm2：
```
开启：pm2 start ***.js		                    # node关闭窗口仍然运行，且会自动重启
参数启动：pm2 start **.js --name=”test” 		 # 启动应用程序并命名为 "api"
关闭：pm2 stop **.js
重启：pm2 restart **.js
```

pm2可以监控并管理多个应用程序，并对其进行日志监控。
